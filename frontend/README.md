# React Frontend for Card Validator

## Setup

1. **Install dependencies:**
```bash
cd frontend
npm install
```

2. **Start development server:**
```bash
npm run dev
```

3. **Open browser:**
```
http://localhost:3000
```

## Backend Required

Make sure Go backend is running on port 8080:
```bash
cd ..
go run main.go
```

## Build for Production

```bash
npm run build
```

Output will be in `dist/` folder.

## Features

- ✅ React 18
- ✅ Vite for fast development
- ✅ Modern hooks (useState, useEffect)
- ✅ Component-based architecture
- ✅ Responsive design
- ✅ Dark mode toggle
- ✅ Floating animations
- ✅ API integration with Go backend
